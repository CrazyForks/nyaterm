# 快捷键

Dragonfly 提供丰富的快捷键支持，提升操作效率。

## 终端操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+C` | 复制选中文本 / 发送中断信号 |
| `Ctrl+V` | 粘贴 |

## 标签与会话

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+N` | 新建 SSH 会话 |
| `Ctrl+Shift+T` | 新建本地终端 |
| `Ctrl+W` | 关闭当前标签 |
| `Ctrl+Tab` | 下一个标签 |
| `Ctrl+Shift+Tab` | 上一个标签 |
| `Ctrl+1~9` | 切换到标签 1-9 |

## 视图与布局

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+B` | 切换左侧边栏 |
| `Ctrl+Shift+B` | 切换右侧边栏 |
| `Ctrl++` | 放大 |
| `Ctrl+-` | 缩小 |
| `Ctrl+0` | 重置缩放 |
| `F11` | 切换全屏 |

## 特殊功能

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+,` | 打开设置 |
| `Ctrl+L` | 锁定屏幕 |

:::tip 提示
快捷键可能因操作系统不同而有所差异。macOS 上 `Ctrl` 对应 `Cmd` 键。
:::
