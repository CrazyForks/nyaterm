# 终端功能

Dragonfly 基于 xterm.js 提供了功能丰富的终端模拟器。

## 终端渲染

### 硬件加速

默认启用 WebGL 渲染，利用 GPU 加速终端绘制，确保流畅的滚动和输出体验。

可在 **设置 → 终端 → 硬件加速** 中开启或关闭。

### 字体配置

- **字体系列** — 支持多字体回退链，首选字体优先级最高
- **字体大小** — 终端和界面字体大小分别可调
- **字体连字** — 可启用编程字体连字特性

内置 JetBrains Mono 和 Noto Sans SC 字体，也可使用系统安装的任意字体。

### 光标样式

支持三种光标样式：
- 块（Block）
- 下划线（Underline）
- 竖线（Bar）

可选择是否启用光标闪烁。

## 终端操作

### 右键菜单

在终端中右键可以：

- 复制 / 粘贴
- 粘贴选定的文本
- 查找文本
- 在线搜索选中文本
- 翻译选中文本
- 清屏 / 全部清除
- 全选

### 回滚缓冲区

终端保留可配置行数的历史输出（默认 1000 行），可通过滚轮向上滚动查看历史内容。

在 **设置 → 终端 → 回滚缓冲区** 中调整行数。

### 剪贴板

- **选中即复制** — 选中文本自动复制到剪贴板
- **右键粘贴** — 右键点击终端区域粘贴剪贴板内容

在 **设置 → 交互** 中配置。

## 关键词高亮

Dragonfly 支持终端文本的关键词高亮：

### 内置规则

自动识别并高亮常见模式，包括：
- 日期时间格式
- 数字
- 错误/警告关键词

内置规则会根据当前主题自动调整配色。

### 自定义规则

在 **设置 → 终端 → 关键词高亮** 中添加自定义规则：

1. 设置规则名称
2. 选择高亮颜色
3. 添加匹配模式（支持正则表达式），每行一个

自定义规则的优先级高于内置规则。

## 命令历史

终端会自动记录输入的命令：

- 按会话分组存储
- 支持模糊搜索
- 在左侧边栏的 **命令历史** 面板中查看

## 联想补全

终端输入时会自动显示命令联想补全建议，基于历史命令进行匹配。

## Keep-Alive

可配置 SSH Keep-Alive 间隔，防止连接因空闲而断开：

- 在 **设置 → 终端 → Keep-Alive 间隔** 中设置
- 设为 0 则禁用

## 远程资源监控

在 **设置 → 终端** 中启用 **显示远程资源信息** 后，状态栏会显示当前 SSH 主机的 CPU 和内存使用情况，每 10 秒更新一次。
