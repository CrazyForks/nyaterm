# 主题与外观

Dragonfly 支持高度可定制的界面外观。

## 主题切换

在 **设置 → 外观 → 主题** 中选择颜色主题。支持深色和浅色模式。

也可以通过菜单 **视图 → 主题** 快速切换。

## 界面缩放

- **放大** — `Ctrl++` 或菜单 **视图 → 放大**
- **缩小** — `Ctrl+-` 或菜单 **视图 → 缩小**
- **重置** — `Ctrl+0` 或菜单 **视图 → 重置缩放**

## 面板布局

界面采用三栏布局，每个面板都可以调整大小：

- **左侧边栏** — 已保存的连接、活动会话、命令历史
- **中央区域** — 终端标签页
- **右侧边栏** — 文件浏览器、文件传输、快捷命令

可通过菜单 **视图 → 重置面板布局** 恢复默认布局。

## 全屏模式

按 `F11` 或通过菜单 **视图 → 全屏** 进入全屏模式。

## 语言设置

在 **设置 → 外观 → 语言** 中切换界面语言，目前支持：

- 简体中文
- English

## 字体配置

在 **设置 → 外观** 中配置字体：

- **字体系列** — 选择终端和 UI 字体，支持多级回退
- **终端字体大小** — 终端文字大小（像素）
- **界面字体大小** — UI 文字大小（像素）

内置字体包括 JetBrains Mono 和 Noto Sans SC，也会列出系统上已安装的字体供选择。
