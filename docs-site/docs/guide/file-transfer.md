# SFTP 文件传输

Dragonfly 内置了功能完善的 SFTP 文件管理器，让你无需额外工具即可管理远程文件。

## 文件浏览器

连接 SSH 会话后，右侧边栏会显示远程文件浏览器：

- 自动定位到用户主目录
- 点击文件夹进入，点击上方 **返回上级** 按钮返回
- 点击路径栏可直接输入路径跳转
- 支持自动同步终端当前路径

### 文件操作

右键点击文件或文件夹，可执行以下操作：

| 操作 | 说明 |
|------|------|
| 打开 | 在默认编辑器中打开文件 |
| 下载 | 下载文件到本地 |
| 重命名 | 重命名文件或文件夹 |
| 移动 | 将文件移动到指定路径 |
| 删除 | 删除文件或文件夹（递归） |
| 属性 | 查看文件详细信息 |
| 复制路径 | 复制文件的完整路径 |

### 新建文件和文件夹

在工具栏中可以：

- 新建文件
- 新建文件夹
- 新建符号链接

## 文件上传

点击工具栏的 **上传** 按钮，选择本地文件上传到当前目录。

也可以通过右键菜单选择 **上传到当前文件夹**。

## 文件下载

选中文件后，点击工具栏的 **下载** 按钮，或右键选择 **下载**。

Dragonfly 使用管道化传输技术，同时维持多个并发数据块读取，大幅提升大文件传输速度。

## 传输进度

在右侧边栏的 **文件传输** 面板中，可以查看所有传输任务的实时进度：

- 传输中的文件名和进度
- 已完成的传输
- 传输错误

可以一键清除已完成的传输记录。

## 路径同步

文件浏览器支持与终端路径同步：

- **手动同步** — 点击同步按钮，将文件浏览器定位到终端当前目录
- **自动同步** — 开启后，文件浏览器会自动跟随终端的工作目录变化

## 文件属性

右键点击文件选择 **属性**，可查看：

- 文件大小
- 修改时间和访问时间
- 所有者和组
- 权限（用户/组/其他，八进制表示）
- UID 和 GID
