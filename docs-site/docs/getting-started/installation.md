# 安装指南

## 系统要求

Dragonfly 支持以下操作系统：

- **Windows** 10/11 (64-bit)
- **macOS** 12+ (Intel & Apple Silicon)
- **Linux** (Ubuntu 20.04+, Fedora 36+, Arch Linux 等)

## 下载安装

### 从发布页面下载

前往 [Releases](https://git.coderkang.top/Tauri/dragonfly/releases) 页面，下载适合你操作系统的安装包：

| 平台 | 安装包格式 |
|------|-----------|
| Windows | `.msi` / `.exe` |
| macOS | `.dmg` |
| Linux | `.deb` / `.AppImage` |

### 从源码构建

如果你想从源码构建，请参考 [开发环境搭建](../development/setup) 章节。

## 首次启动

安装完成后启动 Dragonfly，你会看到一个干净的界面，包含：

- **左侧边栏** — 已保存的连接列表
- **中央区域** — 终端标签页
- **右侧边栏** — 文件浏览器和快捷命令

接下来，请查看 [快速开始](./quick-start) 了解如何创建你的第一个 SSH 连接。
