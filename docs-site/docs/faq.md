# 常见问题

## 连接问题

### SSH 连接超时怎么办？

1. 检查服务器地址和端口是否正确
2. 确认服务器的 SSH 服务是否正在运行
3. 检查防火墙是否允许 SSH 端口
4. 如果使用代理，确认代理配置是否正确
5. 尝试增加 Keep-Alive 间隔（**设置 → 终端 → Keep-Alive 间隔**）

### 连接后终端没有响应？

- 尝试按 `Enter` 键
- 检查用户认证是否成功
- 查看应用日志（**帮助 → 查看日志**）

### 如何使用私钥连接？

1. 前往 **设置 → 安全 → 密钥管理**
2. 点击 **添加密钥**，导入私钥文件
3. 创建连接时，认证方式选择 **私钥**，然后选择对应的密钥

## 文件传输

### 上传/下载速度慢？

Dragonfly 使用管道化传输优化了大文件传输速度。如果速度仍然不理想，可能是网络带宽限制。

### 无法删除文件？

检查当前用户是否有该文件的删除权限。可在文件属性中查看权限信息。

## 界面问题

### 界面字体显示异常？

1. 前往 **设置 → 外观 → 字体系列**
2. 确认首选字体已正确安装
3. 添加备选字体作为回退

### 终端渲染卡顿？

尝试在 **设置 → 终端** 中：
- 开启或关闭 **硬件加速**（需要重启应用）
- 减少 **回滚缓冲区** 行数

### 如何重置界面布局？

通过菜单 **视图 → 重置面板布局** 恢复默认布局。

## 安全相关

### 忘记了主密码怎么办？

主密码用于加密会话数据。如果忘记，需要删除 `~/.dragonfly/` 目录下的配置文件重新配置。

:::warning 注意
删除配置文件将丢失所有已保存的连接和密钥。
:::

### 忘记了锁屏密码怎么办？

可以手动编辑 `~/.dragonfly/settings.json` 文件，将锁屏相关配置项重置。

## 其他

### 如何导入其他 SSH 客户端的会话？

目前支持从 WindTerm 导入：

1. 右键点击侧边栏选择 **导入会话**
2. 选择 WindTerm
3. 选择 WindTerm 的会话配置文件

### 配置文件在哪里？

所有配置文件存储在 `~/.dragonfly/` 目录下，包括连接、密钥、设置等。

### 如何查看应用日志？

通过菜单 **帮助 → 查看日志** 打开日志文件。日志按天轮转，保留最近 7 天的记录。
