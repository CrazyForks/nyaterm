# Installation

## System Requirements

Dragonfly supports the following operating systems:

- **Windows** 10/11 (64-bit)
- **macOS** 12+ (Intel & Apple Silicon)
- **Linux** (Ubuntu 20.04+, Fedora 36+, Arch Linux, etc.)

## Download & Install

### From Releases

Visit the [Releases](https://git.coderkang.top/Tauri/dragonfly/releases) page and download the installer for your OS:

| Platform | Format |
|----------|--------|
| Windows | `.msi` / `.exe` |
| macOS | `.dmg` |
| Linux | `.deb` / `.AppImage` |

### Build from Source

To build from source, see the [Development Setup](../development/setup) section.

## First Launch

After installation, launch Dragonfly and you'll see a clean interface with:

- **Left Sidebar** — Saved connections list
- **Center Area** — Terminal tabs
- **Right Sidebar** — File explorer and quick commands

Next, check out [Quick Start](./quick-start) to create your first SSH connection.
